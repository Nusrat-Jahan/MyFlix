Objective

This web application will provide users access to different information about movies, directors, genres etc. Additionally, Users will be able to sign up, update their personal information as well as create a list of their favorite movies.

Description

Method

URL

Query Parameters

Body of Request

Body of Response
Return a list of ALL movies to the user GET /movies None None

A JSON object holding the data about all movies
Return data (genre, director, release year) about a single movie by title to the user GET /movies/:title Title None

A JSON object holding data about a single Movie, containing title, genre, release year.

e.g.
{
   "Genre": {
   "Name": "Thriller",
   "Description": "Thriller film, also known as suspense film or suspense thriller, is a broad film genre that involves excitement and suspense in the audience."
},
   "Director": {
    "Name": "Jonathan Demme",
   "Bio": "Robert Jonathan Demme was an American director, producer, and screenwriter.",
   "Birth": "1944",
   "Death": "2017"
},
   "Actors": [],
   "_id": "606f12ec196e5f78e38d4446",
   "Title": "Silence of the Lambs",
   "Description": "Clarice Starling, an FBI agent, seeks help from Hannibal Lecter, a psychopathic serial killer and former psychiatrist, in order to apprehend another serial killer who has been claiming female victims.",
   "ImagePath": "silenceofthelambs.png",
   "ReleaseYear": "1991",
   "ImdbRating": 8.6,
   "Featured": true
}

Return data about a genre (description) by name/title (e.g., “Thriller”) GET /movies/genre/:name Name None

An html response with a description of the requested genre
Return data about a director (bio, birth year, death year) by name GET movies/director/:name Name None

An html response with a description of the requested director
Allow new users to register POST /users None

A JSON object holding data about the new user to add, structured like: {
   Username: "Nusrat"
   Password: ********
   Email: dummy@test.com
   Birthdate: 01/01/1995
   FavoriteMovies: [606f4e01196e5f78e38d4448, 606f4c28196e5f78e38d4447] }

An html response that the account has been created

e.g.
Thank you for subscribing to MyFlix

Allow users to update their user info PUT /users/:username Username $set: {    Username: "Nusrat Updated",
   Password: *****,
   Email: updated@update.com,
   Birthday: 1995/07/07
}

An html response showing the user information has been updated

e.g.
Successfully updated the username.

Allow users to add a movie to their list of favorites POST /users/:Username/movies/:MovieID Username
MovieID

A JSON object holding data about the favorite movie to add, structured like: {
{
   Username: "Nusrat"
   Password: ********
   Email: dummy@test.com
   Birthdate: 01/01/1995
   FavoriteMovies: [606f4e01196e5f78e38d4448]
}

An html response that the movie selected has been added to the favourites list

e.g.
Movie has been added to your favourites!
Allow users to remove a movie from their list of favorites DELETE /users/:Username/movies/:MovieID Username
MovieID		 None

An html response that the movie selected has been removed from the favourites list

e.g.
Movie has been deleted from your favourites!
Allow existing users to deregister DELETE /users/:Username Username None

An html response that the account has been deleted

e.g.
Your account has been deleted!
